## Methodology

1. For each country, load daily scores for conflict, humanitarian, economic, disaster, political
2. For each pair (A, B), compute Pearson correlation at lags 7-30 days
3. A 'leads' B at lag L means: A(t) correlates with B(t+L)
4. Filter for |r| > 0.3 and p < 0.05 (two-tailed)
5. Results inform which dimensions can serve as early-warning signals
