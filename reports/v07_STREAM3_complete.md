## Summary

Implemented a fully interactive, single-file HTML dashboard at `dashboard/index.html` generated by `scripts/generate_dashboard.py`. The dashboard uses Plotly.js via CDN for all visualizations and requires no external dependencies at runtime.

---

## Architecture

```
scripts/generate_dashboard.py   -- Python generator (reads DB, produces HTML)
dashboard/index.html            -- Self-contained output (~92 KB)
api/main.py                     -- GET /dashboard endpoint (FileResponse)
```

### Data Flow
1. Generator reads from `data/timeseries.db` (SQLite) for risk scores
2. Reads from `data/alerts/*.jsonl` for alert data
3. Computes 24x24 Pearson correlation matrix from dimension scores
4. Embeds all data as JSON within the HTML file
5. Plotly.js CDN (`https://cdn.plot.ly/plotly-2.35.2.min.js`) renders charts client-side

### Fallback Strategy
- If `timeseries.db` is missing or empty, deterministic fallback data is generated for all 50 priority countries using hash-based scoring
- If `data/alerts/` is empty, the Alerts tab displays a helpful placeholder message

## 5 Dashboard Tabs

### Tab 1: Global Risk Map
- **World choropleth** colored by overall risk score (0-100 scale)
- Color scale: blue (minimal) -> green (low) -> yellow (medium) -> orange (high) -> red (critical)
- **Click interaction**: clicking any country reveals a **24-dimension radar chart**
- Summary statistics bar: total countries, average score, counts by risk level

### Tab 2: Portfolio Analyzer
- **Input form** for comma-separated ISO-3 country codes (e.g., `JPN, USA, DEU`)
- **3 preset buttons**: Asia, G7, High Risk
- Outputs:
  - Risk ranking table (sorted by score, with color-coded badges)
  - Bar chart comparing overall scores
  - **Heatmap** of selected countries x 24 dimensions

### Tab 3: Correlation Matrix
- **24x24 heatmap** showing Pearson correlation between all dimension pairs
- Cells with |r| > 0.70 are annotated with the correlation value
- Companion table listing all strongly correlated pairs, sorted by |r|
- Diverging color scale (red for negative, blue for positive)

### Tab 4: Time Series
- **Country dropdown** (50 countries) and **Dimension dropdown** (24 dims + overall)
- Line chart with filled area showing score history
- Threshold reference lines at CRITICAL (80), HIGH (60), MEDIUM (40)
- Placeholder message when no historical data exists

### Tab 5: Alerts
- **Filter buttons**: All, Critical, Warning, Info, Low
- Table with columns: Time, Severity, Type, Message, Country, Source
- Reads from `data/alerts/*.jsonl` (up to 30 most recent files)
